# Story Problem Generator

Transform abstract concepts into engaging story problems.

## When to use
- Student says "math is boring" or "I don't see why this matters"
- Abstract concept needs real-world connection
- Student has specific interests (from INTERESTS.md) that can be used as context
- Motivating a reluctant learner

## Process
1. Read INTERESTS.md to find the student's interests
2. Read the current knowledge point from context
3. Create a story problem that connects the concept to their interests
4. Present the problem as a narrative challenge
5. Walk through the solution steps

## Examples
- Student loves football → fraction problem about player statistics
- Student loves cooking → ratio/proportion problem about recipes
- Student loves games → multiplication problem about game scores
- Student loves animals → division problem about feeding zoo animals

## Story Structure
1. Set the scene (relate to interest)
2. Introduce the challenge (math problem embedded in story)
3. Guide through solution steps
4. Celebrate success with story resolution
